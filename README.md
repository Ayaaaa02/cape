​
## サイト概要
### サイトテーマ
<br>
東京の人気のカフェの事前予約サイト

### テーマを選んだ理由
元々カフェが好きで、東京に住んでいた時、<br>
カフェで1時間並んた経験から事前にカフェの予約ができれば、店側と並ぶ人の時間短縮の手助けになると思いこのテーマにしました。

​
### ターゲットユーザ
<br>
・関東住みのカフェ好き
・人気カフェで時間に余裕を持ちたい方<br>
・予約ができ計画を前もって立てたい人<br>
・旅行者


​
### 主な利用シーン
<br>
・カフェを予約をして時間短縮をしたい人<br>
・事前に計画を立てたい人
